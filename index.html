✦ Features

Everything you need.
Nothing you don't.

A clean, powerful workspace that adapts to what you're doing — whether that's writing a report, studying chemistry, or debugging a Node.js app.

Desktop-Native

Not a browser tab. A real app that runs on your machine with file access, shell commands, and offline support. Feels fast because it is.

Multi-Model

Switch between GPT-4o, Claude, Gemini, Copilot, and more. Use your own API keys or sign in with OAuth. No middleman pricing.

Built-in Code Editor

A full workspace with file explorer, syntax highlighting, and an AI that reads, edits, and creates files directly in your project.

Rich Artifacts

The AI creates interactive HTML, Markdown documents, PDFs, spreadsheets, and more — all rendered right in the workspace.

Web Search

The AI can search the web for current information, read documentation, and cite sources — so you get accurate, up-to-date answers.

Private & Secure

Your data stays on your machine. No telemetry, no cloud storage, no account required. API keys are stored locally and never shared.

⚡ Three Modes

One app. Three superpowers.

Switch between modes instantly. Each one is purpose-built with specialized tools, prompts, and workflows.

Work Mode

Turn rough ideas into polished deliverables — plans, reports, briefs, emails, spreadsheets, and presentations. Systematic, execution-focused AI for professionals.

Study Mode

Learn any subject with an AI tutor that adapts to your level. Step-by-step explanations, practice quizzes, flashcards, and visual aids — designed for deep understanding.

Code Mode

An AI coding assistant that reads your repo, writes clean code, runs tests, and verifies its own work. Like pair programming with a senior engineer.
